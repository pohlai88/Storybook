# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-01-03

### Added
- Initial release of AIBOS Design System Storybook
- Web Components support with `@storybook/web-components-vite`
- TypeScript configuration and type definitions
- Production-optimized build configuration
- Code splitting and chunk optimization (25 chunks)
- Accessibility addon integration
- Auto-documentation generation
- Dark theme support
- Responsive design
- Deployment configurations (Vercel, Netlify, GitHub Pages)
- Comprehensive documentation

### Features
- ✅ Storybook 8.6.15 with Vite builder
- ✅ Web Components stories
- ✅ React component support (optional)
- ✅ Design tokens integration
- ✅ CSS framework integration
- ✅ Production build optimizations
- ✅ Bundle size optimization (~2.7 MB total)
- ✅ Performance optimizations

### Documentation
- README with usage examples
- Deployment guide
- Architecture documentation
- Component documentation
